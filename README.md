## Сборка для быстрого старта нового проекта

> Сборка работает на Gulp версии 4.0

#### Используется

* pug
* scss
* настроенный gulpfile
* csscomb.json (для форматирования кода)
* browser-sync
* готовая структура папок и файлов

#### Установка проекта

1. ```clone this repo```
2. ```cd path/to/...```
3. ```npm install gulpjs/gulp-cli -g```
4. ```npm install```  
5. ```gulp```
